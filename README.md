# Halloween-MMO
Репозиторий игры для Furry Games and Dev Jam #4 (SFW).

Изучайте но не используйте этот проект как основу для своих игр или проектов. Не зарабатываайте деньги без разрешения участников этого проекта. Вся музыка, текстуры, ассеты, код - принадлежат нашей команде и не должны использоваться в сторонних проектах без разрешения.<br>
Звуки шагов и подбора предметов были взяты на opengameart и имеют лицензию CC0<br>
Разрешена модификация и улучшение игры, вы можете хостить свои собственные сервера и вносить изменения в основной проект. Я не обещаю быстрое рассмотрение пуллреквестов, но я постараюсь уделить им внимание по мере своих возможностей.

hmmo-client - это папка с годот проектом игрового клиента

hmmo-server - это папка с годот проектом игрового сервера
